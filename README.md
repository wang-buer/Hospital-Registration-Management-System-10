# 医院挂号管理系统10



## 1、项目介绍

医院挂号管理系统10拥有俩角色

管理员：用户管理、挂号管理、统计图管理

用户：添加用户＼挂号

## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：Tomcat 7-10版本均可
- maven项目：否
- 硬件环境：Windows
